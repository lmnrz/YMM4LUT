# YMM4LUT

## 概要

.cube形式のLUTを適用する映像エフェクトです

## 導入方法

1. [ここ](https://github.com/lmnrz/YMM4LUT/releases/ "releases") から最新版の.ymmeファイルをダウンロードする
2. ダウンロードしたファイルをダブルクリックして開き、「このプラグインの開発者を信頼する」のチェックボックスを押してから「インストール」を押す
3. 完了

## 既知の問題点

・他ソフトのLUT実装と比べると高アルファなピクセルの色相に誤差があるかもしれません

修正はいつかします　新しいバグ報告や御指摘があれば気軽に[issues](https://github.com/lmnrz/YMM4LUT/issues "issues")までどうぞ
